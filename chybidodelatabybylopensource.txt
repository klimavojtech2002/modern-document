Open-Source Release Plan: mcp-md-html-pdf

     Context

     Project mcp-md-html-pdf (GitHub: modern-document) is a functional
     Markdown-to-branded-HTML/PDF engine (~1500 LOC). It's published on npm
 but the
      GitHub repo isn't public yet. The goal is to prepare it to Google
senior
     engineer quality before the first public push. Key issues: Czech
language
     throughout source, XSS vulnerability, missing community/CI files, test
     artifacts in root, no tests.
